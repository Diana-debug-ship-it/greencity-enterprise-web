# Green City Real Estate — Международный портал недвижимости (Webflow Enterprise)

Флагманский коммерческий проект — полноценный многостраничный веб-портал для международного агентства недвижимости **Green City Real Estate** (Дубай, ОАЭ). Проект демонстрирует навыки глубокой кастомной разработки на базе Webflow, интеграции облачных NoSQL баз данных, настройки продвинутых интерфейсных библиотек и автоматизации контент-маркетинга.

* **Статус проекта:** В продакшене, является основным цифровым активом компании.
* **Ссылка на сайт:** [greencityre.com](https://greencityre.com/)
* **Инструменты разработки:** Webflow Enterprise CMS, Supabase (PostgreSQL под капотом), JavaScript, Finsweet Components, Swiper.js, REST API.

---

## 🛠 Техническая архитектура и интеграции

Стандартные возможности платформы были существенно расширены за счет кастомного кода и интеграции внешних сервисов:

1. **Облачная база данных (Supabase):** 
   Поскольку стандартная CMS Webflow не умеет динамически сохранять пользовательские действия, я подключила **Supabase**. Через кастомные JavaScript-скрипты реализована логика подсчета и хранения просмотров страниц, а также система оценки (рейтинга) статей в блоге.
2. **Многоязычность (Localization):** 
   Портал полностью адаптирован под международный рынок и поддерживает переключение языков (RU, EN, DE, FR). Логика маршрутизации и подгрузки правильных языковых пакетов настроена на уровне архитектуры страниц.
3. **Автоматизация контента (AI-Driven CMS Pipeline):**
   Реализована система автоматической генерации и выкладки статей. Контент-менеджер создает черновик, система автоматически переводит его на несколько языков, генерирует структуру и отправляет на апрув человеку. После подтверждения статья автоматически публикуется в Webflow CMS через API.
4. **Интерфейсные решения (UI/UX):**
   * Интегрирована библиотека **Swiper.js** для кастомных, производительных и адаптивных слайдеров недвижимости.
   * Использованы компоненты **Finsweet** для продвинутой фильтрации, поиска и сортировки объектов в каталоге без перезагрузки страниц.
